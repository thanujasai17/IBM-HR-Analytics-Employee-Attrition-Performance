# 📊 IBM HR Attrition Analysis (Machine Learning Project)

## 📌 Project Overview
This project focuses on analyzing employee attrition using the IBM HR dataset. The main objective is to identify key factors that influence employee turnover and build a machine learning model to predict attrition.

---

## 🎯 Objectives
- Analyze employee data to understand attrition patterns  
- Identify key factors affecting employee turnover  
- Build a predictive model for attrition  
- Provide insights to improve employee retention  

---

## 🛠️ Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---

## 📂 Dataset
The dataset contains employee-related information such as:
- Age  
- Gender  
- Department  
- Job Role  
- Monthly Income  
- Job Satisfaction  
- Work-Life Balance  
- Attrition  

---

## 🔄 Project Workflow

### 1. Data Preprocessing
- Loaded dataset using pandas  
- Checked missing values and duplicates  
- Cleaned the dataset  

### 2. Exploratory Data Analysis (EDA)
- Calculated attrition rate (~16%)  
- Visualized:
  - Age distribution  
  - Gender distribution  
  - Department distribution  

### 3. Advanced Analysis
- Analyzed attrition based on:
  - Overtime  
  - Job Satisfaction  
  - Work-Life Balance  
  - Monthly Income  

### 4. Feature Engineering
- Converted categorical data using Label Encoding  

### 5. Model Building
- Used Random Forest Classifier  
- Split dataset into training and testing sets  

### 6. Model Evaluation
- Evaluated using:
  - Accuracy  
  - Confusion Matrix  
  - Classification Report  

---

## 📈 Results
- Model achieved good accuracy  
- Key factors influencing attrition:
  - Overtime  
  - Job Satisfaction  
  - Salary  
  - Work-Life Balance  

---

## 📌 Conclusion
Employee attrition is mainly influenced by workload, satisfaction levels, and compensation.  
The machine learning model helps predict attrition effectively and supports decision-making.
